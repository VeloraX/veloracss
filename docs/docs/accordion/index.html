Accordion

vel-src: components/accordion.css

A collapsible content component for grouping related information into expandable sections. Supports open/close state via the vel-open modifier and a flush borderless variant.

ClassProperties
vel-accordionborder, border-radius: xl, overflow: hidden, background: surface-1
vel-accordion-itemborder-bottom between items; last-child removes border
vel-accordion-triggerfull-width button, flex space-between, hover: surface-2 background
vel-accordion-trigger-icon1.25rem icon container, transition: transform 200ms; rotates 180deg with .vel-open
vel-accordion-contentpadded content area, color: muted, line-height: relaxed
vel-accordion-flushborderless transparent variant; removes outer border and radius
