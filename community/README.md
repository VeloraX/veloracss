# VeloraCSS Community Components

A registry of community-built components and blocks using VeloraCSS.
Browse them at https://velorax.github.io/veloracss/community

## How to submit

1. Fork the repository
2. Copy `community/template.json` to `community/submissions/your-component-name.json`
3. Fill in all required fields
4. Make sure your HTML only uses `vel-` classes and `--vel-*` CSS variables
5. Open a pull request — we will review and merge it

## Requirements

- Uses only `vel-` prefixed classes
- Uses only `--vel-*` CSS custom properties (no hardcoded colors)
- Self-contained HTML (no external dependencies)
- Works in both dark mode (default) and light mode (`data-vel-mode="light"`)
